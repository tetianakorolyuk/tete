# TETÉ Portfolio - Content Editing Guide

## Quick Start

### Access the Edit Page
1. Go to: `https://thetestestudio.com/edit/login`
2. Password: `tete2025`

### Edit Projects
1. Click **Expand** on any project to edit details
2. Update: Title, Subtitle, Year, Location, Description
3. Upload new images or paste image URLs
4. Add/edit facts (Focus, Palette, Details, Mood)
5. Click **Save All** to publish changes

### Upload Images
1. In the edit page, click **Upload** next to an image slot
2. Select an image from your computer
3. Wait for upload to complete (preview will appear)
4. Click **Save All** to publish

### Add New Project
1. Click **+ Add Project** button
2. Fill in all details
3. Upload at least 1 image (2 recommended)
4. Click **Save All**

### Reorder Projects
- Use **↑** and **↓** arrows to move projects
- Click **Save All** to publish new order

### Delete Project
1. Click **Expand** on the project
2. Click **Delete** button
3. Confirm deletion
4. Click **Save All**

## Tips

- **Images**: Use high-quality JPGs, recommended 1920x1080 or larger
- **Descriptions**: Keep them concise (2-4 sentences)
- **Facts**: Use short, descriptive values
- **Preview**: Click "View Site" to see changes live

## Contact Form

Form submissions are sent to: `project@thetetestudio.com`

To change the recipient email:
1. Update `CONTACT_EMAIL` in `.env.local` (local) or Vercel Environment Variables (production)

## Technical Support

- Site is built with Next.js
- Hosted on Vercel
- Custom domain: thetestestudio.com
- CMS stores data in `public/content/projects.json`
