How to use
  1. Select a project on the left.
  2. Edit text on the right.
  3. Add photos by dragging them into the image box.
  4. Click Download update zip and send it to Flavio.

If the download fails and you see file:// in the address bar, open the editor from the live website (GitHub Pages). If you need to run it locally, Flavio can start a simple server: 

python3 -m http.server 8000

Then open: http://localhost:8000/editor.html
