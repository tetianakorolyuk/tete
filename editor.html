How to use

  2. Edit text on the right.
  3. Add photos by dragging them into the image box.
  4. Click Download update zip.

If you opened this file locally and the download fails, open it from the live website (GitHub Pages) or run a local server: 

python3 -m http.server 8000

Then open: http://localhost:8000/editor.html
