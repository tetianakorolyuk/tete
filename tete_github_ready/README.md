# tete — GitHub Pages

## Upload
Upload the contents of this folder to the ROOT of your GitHub repo:
- index.html
- /images (folder)

## Turn on GitHub Pages
1) GitHub repo → Settings  
2) Pages  
3) Source: **Deploy from a branch**  
4) Branch: **main** (or master) / **root**  
5) Save

Your site URL will appear in the Pages settings.

## Edit email
Open `index.html` and change:
`const EMAIL = "hello@tete.studio";`
